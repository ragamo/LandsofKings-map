# Voronoi Procedural Map Generator

Work in progress

## Install dependencies

```sh
$ npm install
```

## Build project

```sh
$ npm run build
```

## Dev mode

Runs a local static webserver with watchify on port 8000

```sh
$ npm run serve
```
